Web开发人员的基本工具包

Emmet — the essential toolkit for web-developers. Emmet is a plugin for many popular text editors which greatly improves HTML & CSS workflow

Emmet是一个能够显著提升HTML&CSS编写效率，支持大多数主流文本编辑器和IDE的插件。

# 2、ID和class

1、id

语法 #

div#header

效果

<div id="header"></div>
1

2、class

语法 .

div.title

效果

<div class="title"></div>
1

3、组合

div#header.title

效果

<div id="header" class="title"></div>
1

div.class1.class2.class3

效果

<div class="class1 class2 class3"></div>
1
