# 5、简单工厂模式

23种设计模式之一

手机工厂-生产手机

//手机类
public class Phone {
	private String brand;
	private Integer price;
	
	
	public Phone(String brand, Integer price) {
		super();
		this.brand = brand;
		this.price = price;
	}
	public String getBrand() {
		return brand;
	}
	public void setBrand(String brand) {
		this.brand = brand;
	}
	public Integer getPrice() {
		return price;
	}
	public void setPrice(Integer price) {
		this.price = price;
	}
	
	public void info(){
		System.out.println("手机品牌："+brand + "\n手机价格："+price);
	}
}

//手机工厂-造手机
public class PhoneFactory {
	
	public static Phone getInsetance(String name){
		Phone p = null;
		if("小米6".equals(name)){
			p = new Phone("小米6",1500);
		}else if("Oppo R13".equals(name)){
			p = new Phone("Oppo R13",2500);
		}else if("Huawei Navo".equals(name)){
			p = new Phone("Huawei Navo",3500);
		}else if("Iphone Xs".equals(name)){
			p = new Phone("Iphone Xs",6000);
		}
		return p;
	}

}

@Test
public void test() {
    //用手机工厂生产小米手机
    Phone phone = PhoneFactory.getInsetance("小米6");
    phone.info();

    phone = PhoneFactory.getInsetance("Oppo R13");
    phone.info();
}

# 6、AOP切面编程

  • AOP的目标：让我们可以“专心做事”

  • AOP原理

    • 将复杂的需求分解出不同方面，将散布在系统中的公共功能集中解决

    • 采用代理机制组装起来运行，在不改变原程序的基础上对代码段进行增强处理，增加新的功能

需求: 添加用户的前后，记录日志

  • Dao层

  • 配置增强

    <aop:aspect ref="userServiceLogger">  <!-- 配置切面 -->

    </aop:aspect>
</aop:config>

  • 完善增强方法

在增强方法中可以获取到目标对象的信息(比如目标对象，方法名，参数，返回值等)

//JoinPoint:连接点对象
public void before(JoinPoint jp) {
     log.info("前置增强的方法执行...");
     log.info("调用"+jp.getTarget()+"的"+jp.getSignature().getName()+"，入参是"+Arrays.toString(jp.getArgs()));
 }

public void afterReturning(JoinPoint jp,Object result) {
    log.info("后置增强的方法执行...");
    log.info("调用"+jp.getTarget()+"的"+jp.getSignature().getName()+"，入参是"+jp.getArgs());
    log.info("目标方法的返回值是..."+result);
}

  • 切入点表达式的规则
public * addNewUser(entity.User)
“*”表示匹配所有类型的返回值。

public void *(entity.User)
“*”表示匹配所有方法名。

public void addNewUser(..)
“..”表示匹配所有参数个数和类型。

* com.service.*.*(..)
匹配com.service包下所有类的所有方法。

* com.service..*.*(..)
匹配com.service包及其子包下所有类的所有方法
