实现动态SQL的元素有:if,choose,where,set,trim,foreach

# 1、if

作用: 判断条件是否成立

<select id="findUserByNameAndRoleId" resultType="user">
    SELECT * FROM smbms_user WHERE 1=1
    <if test="uname != null and uname!=''">
    	and userName LIKE CONCAT('%',#{uname},'%')
    </if>

    <if test="urole!= null">
    	and userRole=#{urole}
    </if>
</select>

# 4、trim

  • 功能最强大的动态元素
public class TrimSqlNode implements SqlNode {

  private SqlNode contents;
  private String prefix;  //前缀
  private String suffix;  //后缀
  private List<String> prefixesToOverride = new ArrayList<String>();  //前缀去重
  private List<String> suffixesToOverride = new ArrayList<String>();  //后缀去重
  private Configuration configuration;

  public TrimSqlNode(Configuration configuration, SqlNode contents, String prefix, String prefixesToOverride, String suffix, String suffixesToOverride) {
    this.contents = contents;
    this.prefix = prefix;
    this.prefixesToOverride = parseOverrides(prefixesToOverride);
    this.suffix = suffix;
    this.suffixesToOverride = parseOverrides(suffixesToOverride);
    this.configuration = configuration;
  }
  //...省略其他代码
}

# 6、choose

choose(when、otherwise) 相当于Java中switch语句 当when有条件满足的时候，就跳出choose

如果前面的条件都不成立，使用otherwise(有otherwise的情况下)

  • 语法
<choose>
	<when test ="条件1"> …</when>
	<when test ="条件2"> …</when>
	<when test ="条件3"> …</when>
	…
	<otherwise>…</otherwise>
</choose>
