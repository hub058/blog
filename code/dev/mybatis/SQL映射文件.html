SQL映射文件

MyBatis真正强大的地方就在于SQL映射语句，功能强大，配置简单。

# 1、单参数入参

需求：根据用户名模糊查询用户

# 1、定义接口方法

List<User> findUserByUserName(String username);
1

# 2、SQL映射语句

<select id="findUserByUserName" parameterType="string" resultType="user">
    SELECT * FROM smbms_user WHERE userName LIKE CONCAT('%',#{userName},'%')
</select>
1
2
3

id：接口的方法名

parameterType：参数类型-字串string(别名)类型

resultType：返回结果类型

这里有一个入参，SQL语句中传入参数的位置使用 #{userName}

如果是只有一个简单类型的参数占位符中可以任意写 ，比如#{aaa}或者#{bbb}。

# 3、测试

UserMapper userMapper = session.getMapper(UserMapper.class);
List<User> users = userMapper.findUserByUserName("张");

# 2、ResultMap

使用ResultMap封装

查询结果中的字段名和属性名不一致的时候使用ResultMap进行自定义映射。

<resultMap type="user" id="userlist">
    <！-- property:实体类中的属性名,column:字段名 -->
    <id property="id" column="id" />
    <result property="userRoleName" column="roleName"/>
</resultMap>

<select id="findByUser" parameterType="user" resultMap="userlist">
    SELECT u.*,r.roleName FROM smbms_user u,smbms_role  r
    WHERE u.userRole=r.id AND userName LIKE CONCAT('%',#{userName},'%')
    AND userRole=#{userRole}
</select>
