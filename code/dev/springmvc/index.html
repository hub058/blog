使用 @RequestParam 注解

  1. 如果没有提供参数就要报错了4.0
HTTP Status 400 - Required String parameter 'user' is not present
1

400 错误，请求的字串类型的参数 user- 没有提供

  • 解决方法，在 @RequestParam注解中使用required属性指定参数是非必须的
@RequestMapping("/welcome.html")
public ModelAndView welcome(@RequestParam(value="user",required=false) String username){
    System.out.println("注解的控制器...");
    System.out.println(username);
    return new ModelAndView("welcome");
}

# 5、扩展

java 使用参数返回数据

public class TestMethod {
	
	public static void main(String[] args) {
		
		ArrayList<String> list = new ArrayList<String>();
		list.add("陈世杰");
        
		String index = index(list);
        
		//此时的list已经是改变后的
		for (String string : list) {
			System.out.println(string);
		}
		//上面打印了两个值===陈世杰+孙梦晨
		System.out.println(index);
		//index 还是方法的返回值 welcome
	}
	
	public static String index(List list){
		list.add("孙梦晨");
		//list对象在方法中修改了
		return "welcome";
	}

}
