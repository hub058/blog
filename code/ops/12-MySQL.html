# 1、MYSQL安装

MySQL配置文件的目录

某些情况下，在卸载重装的时候，之前的配置信息没有删除，可能会影响之后的使用。

可以删除掉这个目录 /var/lib/mysql/

rm -rf /var/lib/mysql

  • 在启动mysql的时候，可能出现的问题

[root@localhost zed]# service mysql start Starting MySQL.. ERROR! The server quit without updating PID file (/var/lib/mysql/localhost.localdomain.pid).

  • 解决方案：查看MySQL后台执行的进行，手动结束掉

    ps -A | grep mysql

    [root@localhost zed]# ps -A | grep mysql
 92902 pts/0    00:00:01 mysqld
    1
    2

    92902 就是mysqld后台进程的pid

    kil -9 92902 强制结束进程，然后重启MySQL

    service mysql start

# 2、修改配置文件

拷贝配置文件到/etc/下面

[zed@localhost ~]$ cp /usr/share/mysql/my-small.cnf /etc/my.cnf

# The MySQL server
[mysqld]
