# 1、虚拟机的运行模式

虚拟机运行模式

寄居架构

  • 作为应用软件安装在操作系统上(比如WIN7或者WIN10,称为宿主)
  • 可以在此应用软件上安装多个操作系统(称为客户机)
  • 直接安装在硬件上的系统为宿主
  • 安装在虚拟机中的操作系统称为客户

# 2、VMWare 安装

安装包可以从FTP或者网络上下载

下载完成后，直接双击安装程序，傻瓜式安装。

从网上找一个注册码，输入注册码就可以使用。

如果是土豪的话，建议支持一下正版。

安装完成后，虚拟机会在我们的系统中注册两个网卡。

VMNET1网卡：桥接模式使用的网卡

桥接模式：虚拟机可以连接物理网络，也就是可以连接到宿主机所在的网络。

VMNET8网卡：是NAT模式使用的网卡

NAT模式：虚拟机只能连接自己所在的宿主机。
