# Linux终端

CentOS自带Terminal快捷键

放大字体: Ctrl shift +
减小字体: Ctrl -
 - 这个加减号是字母区域的+-号，不是小键盘上的:
切换上次执行的命令: 上下键(↑ ↓)
清屏: Ctrl L

# 1、安装

  • 问：为什么使用远程工具？
  • 答：由于不能现场操作等种种环境原因需要使用远程操作，可以使用远程的连接工具。
  • 问：好用的远程工具都哪些？
  • 答：SSH、Xshell、FinalShell 、putty等很多
  • 问：怎么安装？
  • 答：傻瓜式安装，一路下一步。

课程中以FinalShell举例，其他工具使用方式类似。

安装完成后，桌面会有个FinalShell图标
