虚拟机中安装Linux操作系统

本文以CentOS6.5为例

# 1、安装CentOS

安装步骤

  1. 文件 - 新建虚拟机
  2. 典型安装 - 下一步
  3. 选择镜像文件 - 下一步
  4. 填写计算机名称、用户名、密码(很重要，一定要记得密码，之后登录使用)
  5. 选填写虚拟机名称和安装位置(建议不要安装在系统盘)
  6. 下一步，完成。
  7. 等待安装完成，中间需要15-20分钟。
  8. 进入登录界面，填写用户名和密码登录。

# 2、虚拟机的基本操作

# 1、启动和停止虚拟机

关闭/打开虚拟机电源 (可以在虚拟机中用命令打开关闭，不建议直接关闭虚拟机的电源) 挂起虚拟机 (保存当前的工作状态，下次可以快速打开虚拟机恢复工作状态) 重启虚拟机

# 2、虚拟机和宿主切换

Ctrl + G (或者虚拟机中单击可以定位到虚拟机) Ctrl + Alt (可以离开虚拟机，定位到宿主机)

# 3、共享文件夹

1、设置共享文件夹

2、添加宿主机中的目录

我们在共享目录里面的文件，宿主机和虚拟机都可以 访问

虚拟机中的访问路径 /mnt/hgfs/共享文件夹的名字

3、虚拟机中访问

/mnt/hgfs/ 从这个位置可以访问到共享文件夹

2、恢复快照

给系统拍摄的快照，类似Windows中的还原点
