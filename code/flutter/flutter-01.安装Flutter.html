# 1、使用镜像

由于在国内访问Flutter有时可能会受到限制，Flutter官方为中国开发者搭建了临时镜像，大家可以将如下环境变量加入到用户环境变量中：

export PUB_HOSTED_URL=https://pub.flutter-io.cn
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
1
2

  • window配置环境变量的方式如下：

    FLUTTER_STORAGE_BASE_URL PUB_HOSTED_URL

上面的环境变量配置是为了国内能正常使用Flutter。

# 5、安装Android Studio

  1. 下载并安装 Android Studio

  2. 运行Android Studio，下载并安装 Android SDK， Android SDK Platform-Tools 以及 Android SDK Build-Tools，这些都是在开发 Android Flutter 应用时所需要的。

# 6、配置 Android 模拟器

在 Android Virtual Device Manager 中，选择一个或多个你想要模拟的 Android 版本的系统镜像，然后选择 Next 选项。推荐选择 x86 或者 x86_64 镜像。：

由于网络原因，Android Studio相关的组件在下载和安装的时候回比较慢。

需要耐心等待！

# 7、配置开发工具

  • 后台开发人员推荐使用IDEA(比较熟悉)

  • 前端开发人员推荐使用Visual Studio Code

两个工具下载安装完成后，都需要再安装Flutter和Dart插件(Flutter的开发语言，类似JavaScript)。
