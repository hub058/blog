# 你离想要的生活，只差一个野心

来这个城市十年了，我也从青年变成了中年，单我们从什么时候开始，失去了野心，又从什么时候开始，连有野心的人都不相信，却要嘲笑一番了呢？

我喜欢和有野心的人称为朋友，无论他们年龄大小，都必定是有趣的一群人。

美国作家拿破仑-希尔在《思考致富》里说：”无形的意念回带来财富，凡是你心里所想，并且相信的，最终必然能够实现。“

《秘密》一书的作者也说：“你朝思暮想的，终将会被你吸引。”

自言：

所爱的人，

想要的生活，

财富自由。

危险的思想

人连欲望都没了，野心都不剩了，只剩下舒适，舒适区呆久了，人就废了。

我们不得不承认，这世界是分阶层的。

但我们不一样，我们有无数的上升通道，只要你足够有野心，足够敏感，足够努力，你总能看到别样的希望。

自言：

尤其在这个信息，和人工智能的领域，我正站在一个风口浪尖。

你的才华能力素养一定要配得上你的野心。

因为只有更好的自己，才能配得上更好的生活。

# 你讲的话，可能就决定了你的一生

语言，真是伟大。

# 你怎么过一天，你就怎么过一年

坚持做一件事情不累，累的是浑浑噩噩地生活。

比如：每天早上六点起床早读。

::: info 坚持，一种可以养成的习惯

日本作家古川在《坚持，一种可以养成的习惯》中写道：“一旦大脑认定某种行为跟往常一样，就会拼命的维持这种行为。而习惯就是把重复的行动化为无意识的行动”。

:::

# 把事情做到极致，钱不过是身外之物

千万不要着急变现，因为我们这群人是专才，要把事情做好。要明白，只要把事情做得足够好，足够极致，钱不过是身外之物，会滚滚而来。

接下来一定要不忘初心，把内容做好，做到极致，刀座不可替代，哪怕一开始不赚钱。

可是，你会发现一个很有趣的现象：那些总是盯着钱看的人，往往总是赚不到钱，想反，盯着目标，把事情做好，资本自然会找过来。

德国作家博多.舍费在《财务自由之路》中说：”不要当一台赚钱的机器，要拥有一台自动为你赚钱的机器。“

::: info 饲养的金鹅越多，它们下的金蛋越多

对于刚毕业的大学生和创业者来说，好的产品和无可替代的技能，就是为你赚钱的那台机器和那只金鹅。

:::

比如我的一个作家朋友，一看到微博火了就入驻微博，微信火了就入驻微信，后来他光入驻了，忘记了自己立足的本质是写出好文字。好的文字无论放在哪个平台，都不会差。
